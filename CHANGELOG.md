# 更新日志

本项目的所有重要更改都将记录在此文件中。

## [1.0.4] - 2025-12-25

### 工程化
- 🔧 **代码规范**：添加 ESLint 配置和脚本

### 修复
- 🐛 **日志引用**：修复遗漏 logInfo 引用的问题

## [1.0.3] - 2025-12-25

### 优化
- 🧪 **测试覆盖**：新增单元测试，提升代码稳定性

## [1.0.2] - 2025-12-25

### 优化
- 🎨 **日志优化**：全面替换 `console` 输出为统一的 `logger` 工具，规范日志格式
- 🧪 **测试覆盖**：新增 Logger 单元测试，补充 Download 核心流程测试，提升代码稳定性

## [1.0.1] - 2025-12-25

### 修复
- 🐛 **资源注册修复**：修改资源注册方式，使用 `ResourceTemplate` 包装资源 URL，解决 SDK 兼容性问题

## [1.0.0] - 2025-11-22

### 新增
- 🔍 **智能解析**：自动识别分享链接，提取无水印视频地址
- 📥 **一键下载**：支持直接下载无水印视频文件到本地
- 📊 **进度显示**：控制台实时显示下载进度
- 🗑️ **文件管理**：自动管理下载文件，支持自定义工作目录
- 🔌 **协议支持**：完整支持 Model Context Protocol (MCP) 标准
